# ForkGlowy

An attempt at building task management app on firebase backend

**Try at:** [forkglowy.web.app](https://forkglowy.web.app)

## Stack

- **UI :** React JS
- **State Management :** Redux Toolkit
- **Backend :** Google Firebase
- **CSS/ Components :** Bootstrap 5
- **Router :** Wouter
- **Builder :** Vite

## Roadmap
